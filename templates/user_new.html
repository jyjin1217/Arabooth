* 고객(User) 데이터베이스를 수정하려면?

고객의 개별 정보를 직접 입력하거나 양식에 맞는 csv 파일을 선택한 후 추가/수정 혹은 삭제 버튼을 클릭합니다.


* CSV 파일 ?

Comma-Separated Variables의 약자로 필드를 쉼표(,)로 구분한 텍스트 파일입니다.


* CSV 파일 생성 방법

Excel 문서 작성 >> 다른 이름으로 저장 (다른 형식 선택) >> 파일 형식 드롭다운 메뉴에서 CSV 선택 후 저장 >> '예' 선택


* Excel 문서 양식

Excel 문서: sample.xlsx  

작성법: 필드 '서비스타입'은 반드시 채우지 않아도 됩니다. 필드 '서비스타입'이 '퇴실회원'으로 되어있으면 해당 회원은 업로드되지 않습니다.그 외 필드는 모두 채워져 있어야만 합니다.
